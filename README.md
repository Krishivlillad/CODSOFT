# CODSOFT UI/UX Internship Projects

This repository contains the UI/UX design projects completed as part of the **CODSOFT Internship** using **Figma**.
The goal was to design user-friendly, visually appealing, and responsive interfaces across different domains.

---

## ✅ Projects Included

### 1. 📱 Mobile App Signup Flow

* A clean and modern sign-up and login flow for a mobile application.
* Focused on simplicity, personalization, and error-free UX.
* 🔗 [View Figma Design](https://www.figma.com/design/StNx8DzY2EzNn2lWmzEyTM/Sign-In---Sign-UP?m=auto&t=AVRbdwPvynhdre0t-1)

### 2. 🍽️ Restaurant Menu UI

* A digital restaurant menu interface.
* Categorized, easy to navigate, and designed to boost engagement.
* 🔗 [View Figma Design](https://www.figma.com/design/VXxXKV28Gp1C8lL2Ilk9kW/Menu?m=auto&t=AVRbdwPvynhdre0t-1)

### 3. 🛒 E-commerce Website UI

* A sleek and organized e-commerce store layout.
* Features intuitive navigation, categorized products, and responsive layout.
* 🔗 [View Figma Design](https://www.figma.com/design/6ponMy8OLPMC8sRq3peDTJ/WebPage1?m=auto&t=AVRbdwPvynhdre0t-1)
